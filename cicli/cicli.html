Iterazione

Ci sono due tipi di iterazione:

  • for (ciclo definito).
  • while e do while (ciclo indefinito).

Definizione:

  • Ãˆ una struttura di controllo che permette di ripetere un blocco di codice.

Sintassi:

  • for:
    •                         for (valore di inizio ; condizione ; passo) {
    /*Istruzioni da eseguire*/
}
  • Esempio:
    •                         for (int i = 0;i < var;i++) {
    /*Istruzioni da eseguire*/
}
  • while:
    •                         while (condizione) {
    /*Istruzioni da eseguire*/
    /*Passo (necessario per evitare un ciclo infinito)*/
}
  • Esempio:
    •                         while (i < var) {
    /*Istruzioni da eseguire*/
    i++;
}
  • do while:
    •                         for (valore di inizio ; condizione ; passo) {
    /*Istruzioni da eseguire*/
}
  • Esempio:
    •                         do {
    /*Istruzioni da eseguire*/
    /*Passo (necessario per evitare un ciclo infinito)*/
} while (condizione);
